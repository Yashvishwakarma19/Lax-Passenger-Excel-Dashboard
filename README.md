📊 Passenger Traffic Analysis at LAX Airport (Excel Dashboard)
🚀 Project Overview

This project focuses on analyzing passenger traffic data at Los Angeles International Airport (LAX) using Microsoft Excel. The aim was to transform raw airport data into a simple, interactive dashboard that highlights key travel trends across different terminals and categories.

The project demonstrates how Excel can be used as a powerful data analysis and visualization tool without requiring advanced programming.

🎯 Objectives
Analyze passenger traffic trends over time
Compare arrivals vs departures
Understand domestic vs international travel patterns
Identify high-traffic terminals
Build an interactive Excel dashboard using pivot tables and charts

📂 Dataset
Source: Open Data Portal (Los Angeles Government)
Dataset: Passenger Traffic at LAX Airport

Key Attributes:
Report Period (Month/Year)
Terminal
Arrival / Departure
Domestic / International
Passenger Count
🛠️ Tools & Techniques
Microsoft Excel
Pivot Tables
Pivot Charts
Data Filtering & Slicers
Basic Data Visualization

🔍 Key Features
📊 Interactive Dashboard
User-friendly layout
Easy filtering using pivot tools
Clear visual representation of trends

📈 Visualizations Used
Line Chart → Monthly passenger trends
Bar Chart → Terminal-wise traffic
Pie Chart → Arrival vs Departure
Donut Chart → Domestic vs International distribution

🧠 Key Insights
Terminal T5 recorded consistently high passenger traffic
Departures slightly exceeded arrivals
Domestic travel dominated compared to international
Passenger traffic showed clear monthly variation trends

🧹 Data Handling
Dataset was already clean
Directly used in Excel without preprocessing
Focus was on analysis and visualization

This project highlights how Microsoft Excel can be effectively used for data analysis and dashboard creation, providing meaningful insights from real-world datasets in a simple and efficient way.
